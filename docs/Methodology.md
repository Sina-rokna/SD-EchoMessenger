# Methodology and Team Structure

## Selected Methodology: Scrum

The selected software development methodology for this project is
**Scrum**.

Scrum is an agile software development framework based on iterative
development. Since the project requires the system to be developed
through iterations with a maximum duration of **14 days**, Scrum
naturally provides the best fit for this constraint by organizing the
work into short-term **Sprints**.

This approach enables our team to effectively manage changing
requirements and, at the end of each iteration, deliver a comprehensive
progress report along with functional and deliverable parts of the
product to the project stakeholders.
